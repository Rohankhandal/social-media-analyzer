# Social Media Content Analyzer

A smart tool that helps you boost your social media engagement by analyzing your content and providing actionable suggestions.

## What This Project Does

- **Upload Content**: Drag and drop your PDFs and images
- **Text Extraction**: Automatically reads text from PDFs and images
- **AI Analysis**: Get smart suggestions to improve your social media engagement
- **Results Display**: Clean, easy-to-understand insights

## Local run

### Backend Setup
```bash
cd backend
npm install
mkdir uploads
npm run dev
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

### How to Start
1. Start both the backend and frontend
2. Open your browser to the frontend URL.
3. Upload the sample PDF from the `docs/` folder
4. See your content analysis and engagement tips!
